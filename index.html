Objective 1

1. Design and complete robotic and embedded systems solutions that apply to real-world situations and challenges.

Harvest Hawk

Harvest Hawk meets this objective because it was developed as a complete robotics solution for a real agricultural monitoring problem. The system was designed to sense, collect, and represent useful environmental information in a way that could support real-world agricultural analysis. Because the project moved from concept to implementation and functioned as a practical engineering solution, it demonstrates my ability to design and complete an embedded robotics system for a realistic challenge.

Automated Page Turner

The Automated Page Turner meets this objective by solving a real accessibility challenge with a complete embedded-mechanical solution. It was developed as a functioning assistive device rather than as a concept alone, demonstrating my ability to turn a real-world problem into a completed embedded system.

Objective 2

2. Implement a simple microprocessor using digital logic design.

Automated Page Turner

The Automated Page Turner meets this objective by using a microcontroller to execute the timing and sequence required to actuate the page-turning mechanism. The controller takes system logic and turns it into precise repeated motor behavior, which reflects practical implementation of microprocessor-driven control.

Python Plane Classifier

The Python Plane Classifier supports this objective by receiving dimensional inputs, processing them through structured classification logic, and producing a repeatable output. Although implemented in Python rather than hardware logic gates, it still reflects processor-style evaluation of inputs and outputs in a controlled decision system.

Objective 3

3. Demonstrate embedded system design skills, including, but not limited to, microcontroller selection, schematic design, printed circuit board layout, design for electromagnetic compatibility and design for manufacturing.

Automated Page Turner

The Automated Page Turner meets this objective through the embedded design decisions required to make the prototype function reliably. The project involved selecting an appropriate controller, integrating motor-driving hardware, and designing the system so the electronics and mechanics could operate together as a realistic prototype. This reflects embedded system design thinking beyond just writing software.

Pipe Viper

Pipe Viper meets this objective because it required embedded and electrical design in a robotic system with strict physical limitations. Electrical planning, subsystem integration, and component arrangement all had to be handled in a way that supported the overall build and its intended inspection use.

Objective 4

4. Apply knowledge of transducers, actuators and simultaneous hardware and software development in the design of an embedded system.

Automated Page Turner

The Automated Page Turner meets this objective by combining actuators, embedded control logic, and physical mechanism design into one working system. The hardware and software had to be developed together so the mechanism could move consistently and reliably.

Pipe Viper

Pipe Viper meets this objective by integrating sensors, camera systems, electrical hardware, and control behavior into a functional robot. The design required understanding how physical components interact with software-driven operation in order to produce useful inspection behavior.

Objective 5

5. Design and analyze real-time embedded systems, including advanced digital logic design, signal processing and highspeed digital systems.

Harvest Hawk

Harvest Hawk meets this objective through real-time embedded sensing and data collection. The system continuously gathers environmental data such as temperature, humidity, pressure, and light intensity, requiring the controller to process incoming information and maintain reliable operation as conditions change.

Portfolio Website

The Portfolio Website supports this objective by functioning as an interactive system where user input directly changes navigation, layout behavior, and content display. The responsive design and interface logic demonstrate system organization and dynamic response in a reliable front-end environment.

Objective 6

6. Implement and evaluate algorithms and methods enabling autonomy in a mobile robot

Harvest Hawk

Harvest Hawk meets this objective by incorporating automated sensing and embedded decision-making methods that support autonomy-oriented robotic behavior. While the project is more focused on monitoring than autonomous navigation, it still applies logic-based system behavior driven by environmental inputs.

Python Plane Classifier

The Python Plane Classifier supports this objective by applying algorithmic decision-making to automatically classify aircraft based on dimensional input data. Once the model is trained, it evaluates incoming values and produces predictions without manual sorting, demonstrating automated logic and intelligent system behavior.
