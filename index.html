Objective 1

1. Design and complete robotic and embedded systems solutions that apply to real-world situations and challenges.

Harvest Hawk

Harvest Hawk meets this objective because it was developed as a complete robotics solution for a real agricultural monitoring problem. Rather than remaining a concept, the project moved through system planning, hardware integration, embedded development, and implementation as a practical solution intended to collect useful environmental data. The project shows that I can identify a real-world need, design an embedded robotic system around that need, and guide the project through completion.

Automated Page Turner

The Automated Page Turner meets this objective by solving a real accessibility problem with a finished embedded-mechanical system. It required integrating control logic, actuation, and mechanical design into one complete device that performs a useful real-world task. The project demonstrates my ability to develop an embedded solution that applies engineering design to a specific human need.

Objective 2

2. Implement a simple microprocessor using digital logic design.

Automated Page Turner

The Automated Page Turner meets this objective by using microcontroller-driven logic to coordinate timing, motion sequence, and repeatable actuation. The controller interprets inputs and executes defined outputs in a structured way, which reflects practical digital logic implementation in an embedded system. The project shows processor-based control being used to make the system behave consistently and predictably.

Harvest Hawk

Harvest Hawk meets this objective by using microcontroller logic to manage sensing behavior and embedded data collection. The system depends on processor-based control to read inputs, organize system behavior, and support automated monitoring functions. This demonstrates practical implementation of microprocessor logic within a working engineering system.

Objective 3

3. Demonstrate embedded system design skills, including, but not limited to, microcontroller selection, schematic design, printed circuit board layout, design for electromagnetic compatibility and design for manufacturing.

Automated Page Turner

The Automated Page Turner meets this objective through the embedded design decisions required to make the prototype function reliably. The project involved selecting an appropriate controller, integrating motor-driving hardware, and designing the system so the electronics and mechanics could operate together as a realistic prototype. This reflects embedded system design thinking beyond just writing software.

Pipe Viper

Pipe Viper meets this objective because it required embedded and electrical design in a physically constrained robotic platform. The project involved careful decisions about subsystem integration, electrical layout, component placement, and practical system packaging so the robot could function inside tight inspection environments. That demonstrates applied embedded system design in a realistic robotics context.

Objective 4

4. Apply knowledge of transducers, actuators and simultaneous hardware and software development in the design of an embedded system.

Automated Page Turner

The Automated Page Turner meets this objective by combining actuators, embedded logic, and physical mechanism design into one working system. The motors function as actuators while the controller manages their behavior through software. Because the device only works when both the hardware and software are designed together, the project demonstrates simultaneous embedded development across both domains.

Pipe Viper

Pipe Viper meets this objective by integrating sensors, camera systems, electrical hardware, and control behavior into a functional robot. The design required understanding how physical components interact with software-driven operation in order to produce useful inspection behavior. This reflects applied knowledge of transducers, actuators, and coordinated hardware/software design.

Objective 5

5. Design and analyze real-time embedded systems, including advanced digital logic design, signal processing and highspeed digital systems.

Automated Page Turner

The Automated Page Turner meets this objective by relying on timing-sensitive embedded behavior to actuate the mechanism correctly and repeatedly. The system had to respond in the correct sequence and with consistent timing to perform the task successfully. That makes the project a practical example of real-time embedded behavior and performance refinement.

Harvest Hawk

Harvest Hawk meets this objective by using embedded sensing and data collection that depend on coordinated timing and real-time system response. The project processes environmental inputs and generates usable outputs through ongoing embedded operation. This demonstrates real-time embedded behavior where responsiveness and data flow matter to system functionality.

Objective 6

6. Implement and evaluate algorithms and methods enabling autonomy in a mobile robot

Harvest Hawk

Harvest Hawk meets this objective by applying automated sensing and embedded decision-making methods that support autonomous system behavior. Although the project focuses on agricultural monitoring rather than full autonomous navigation, it still requires logic-based behavior driven by environmental inputs. Evaluating the system through collected output data demonstrates how embedded methods can support autonomy-oriented robotics.

Pipe Viper

Pipe Viper meets this objective by using sensing, embedded logic, and coordinated robotic subsystem behavior to support movement and inspection in a physical environment. The project required thinking about how the system could interpret conditions and perform effective robot behavior under constraints. This reflects implementation of methods that contribute to autonomy-oriented operation.
